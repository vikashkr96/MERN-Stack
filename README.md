# MERN Stack 🚀

A structured repository for learning and building full‑stack web applications using the **MERN Stack** — **MongoDB, Express.js, React.js, and Node.js**. This repository documents concepts, practice code, and projects created while learning MERN.

---

## 📌 Purpose

This repository is created to:

* Learn full‑stack web development using MERN
* Understand frontend and backend integration
* Practice REST APIs and database operations
* Build real‑world MERN projects
* Strengthen portfolio and job‑ready skills

---

## 🧠 Tech Stack

### 🔹 Frontend

* React.js
* HTML5
* CSS3
* JavaScript (ES6+)
* Axios / Fetch API

### 🔹 Backend

* Node.js
* Express.js
* RESTful APIs

### 🔹 Database

* MongoDB
* Mongoose
* SQL

### 🔹 Tools & Others

* Git & GitHub
* Postman
* VS Code
* npm

---

## 🛠 Requirements

* Node.js (v16+ recommended)
* npm or yarn
* MongoDB (local or Atlas)

Install dependencies:

```bash
npm install
```

---

## 🚀 How to Run a Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
