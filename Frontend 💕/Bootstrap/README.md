## ğŸ§© Main Components of Bootstrap

### 1. Grid System
- Uses a **12-column layout**
- Helps in creating responsive layouts
- Classes like: `container`, `row`, `col-*`

### 2. Typography
- Predefined styles for:
  - Headings
  - Paragraphs
  - Lists
  - Text utilities (alignment, color, weight)

### 3. Buttons
- Ready-made button styles:
  - `btn`
  - `btn-primary`
  - `btn-success`
  - `btn-danger`, etc.

### 4. Forms
- Styled input fields
- Checkboxes and radio buttons
- Form validation support

### 5. Components
Reusable UI elements such as:
- Navbar
- Cards
- Modals
- Alerts
- Badges
- Carousel
- Dropdowns
- Pagination

### 6. Utilities
- Helper classes for:
  - Spacing (`m-`, `p-`)
  - Colors
  - Display
  - Flexbox
  - Visibility

### 7. JavaScript Plugins
Interactive components like:
- Modal
- Tooltip
- Collapse
- Dropdown
- Toast
